¿Qué es el FRONT-END?

El frontend es la parte de un sitio web o aplicación que interactúa directamente con los usuarios. Incluye todo lo relacionado con la interfaz de usuario, como el diseño, la disposición, la navegación y los elementos interactivos.

Fundamentos del FRONT-END

  • HTML: Lenguaje que define la estructura de una página web.
  • CSS: Controla la apariencia y el diseño de los elementos.
  • JavaScript: Permite interactividad y dinamismo en una web.
  • DOM: Representación estructurada de un documento HTML.
  • Responsive Design: Adapta el diseño a diferentes dispositivos.
  • Preprocesadores CSS: SASS, LESS para mejorar la escritura de CSS.
  • Versionamiento con Git: Control de cambios en el código.
  • Optimización del rendimiento: Minimizar archivos y mejorar carga.
  • Accesibilidad web (A11Y): Mejores prácticas para todos los usuarios.

Frameworks y Librerías Populares

  • React: Biblioteca de JavaScript para construir interfaces de usuario.
  • Vue.js: Framework progresivo para la creación de interfaces.
  • Angular: Framework completo para el desarrollo frontend.
  • Bootstrap: Biblioteca CSS para diseño responsivo.
  • Tailwind CSS: Framework de CSS para diseño rápido y personalizado.
